# Maude Skills

Claude Code skills that complement the Maude framework. Install by copying to `~/.claude/skills/`.

## Maude — Configuration Authority

> *"I don't lose files. Files don't get lost when I'm around."*

Maude watches the watchers. She audits your Room configurations for drift, manages session persistence across memory tiers, delivers morning briefings, and always knows where everything is — because she put it there.

**Install:**
```bash
cp -r skills/maude ~/.claude/skills/
```

**Use:** `/maude sweep`, `/maude brief`, `/maude save`, or just ask her anything about your setup.

Maude is an original character by John Broadway — not from the films, but very much part of Maude.
